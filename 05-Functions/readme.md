# 📊 SQL Functions Practice – Numeric, String & Date Functions

## 📌 Project Overview
This project focuses on practicing important **SQL built-in functions** used for **data transformation, cleaning, and analysis**.

The objective of this exercise was to understand how SQL handles different types of operations, including numeric calculations, string manipulation, and date analysis using a sample employee dataset.

These functions are widely used by **Data Analysts and Data Scientists** when preparing datasets for reporting, analysis, and dashboards.

# 🧮 Numeric Functions

Numeric functions perform **mathematical operations on numerical values**.

### Functions Covered

- **SQRT()** – Calculates the square root of numeric values  
- **ROUND()** – Rounds numbers to a specified precision  
- **RAND()** – Generates random numbers  

### Practical Applications

These functions are commonly used in:

- Mathematical calculations  
- Financial analysis  
- Data normalization  
- Random sampling and simulations  

---

# 🔤 String Functions

String functions are used to **manipulate and transform textual data**.

They are especially useful in **data cleaning and preprocessing tasks**.

### Functions Practiced

- **ASCII()** – Returns ASCII value of the first character  
- **CHAR()** – Converts ASCII code into a character  
- **CHARINDEX()** – Finds the position of a substring  
- **LEFT()** – Extracts characters from the left side of a string  
- **RIGHT()** – Extracts characters from the right side  
- **LEN()** – Calculates the length of a string  
- **LOWER()** – Converts text to lowercase  
- **UPPER()** – Converts text to uppercase  
- **SUBSTRING()** – Extracts a portion of a string  
- **REPLACE()** – Replaces specific text within a string  
- **REVERSE()** – Reverses characters in a string  
- **STUFF()** – Inserts characters into a string  
- **CONCAT()** – Combines multiple strings together  

### Practical Applications

These functions help in:

- Extracting **email domains**
- Cleaning inconsistent text data
- Formatting strings
- Combining columns for reporting

---

# 📅 Date Functions

Date functions allow working with **time-based data**, which is essential for business analytics and reporting.

### Functions Covered

- **GETDATE()** – Retrieves the current system date and time  
- **DATEADD()** – Adds time intervals to a date  
- **DATEDIFF()** – Calculates the difference between two dates  
- **DATEPART()** – Extracts a specific part of a date  
- **DAY()** – Extracts the day from a date  
- **MONTH()** – Extracts the month from a date  
- **YEAR()** – Extracts the year from a date  
- **CONVERT()** – Converts date formats  
- **FORMAT()** – Formats date values into readable styles  

### Practical Applications

These functions are useful for:

- Calculating **employee experience**
- Determining **working duration**
- Extracting **joining month or year**
- Formatting dates for reports and dashboards
